# site_amulherdocomputador
Espaço Democrático e Tecnológico feito por mulheres
A intenção é oferecer um portal tecnológico de serviços para MULHERES, incentivando a alfabetização tecnológica e empodeirando cada vez mais mulheres em atuações empreendedoras e potencializando a divulgação de oportunidades de participação da comunidade digital
